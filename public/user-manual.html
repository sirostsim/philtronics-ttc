What is this system?

The Time to Complete system is used on the shopfloor to record how long it takes to assemble each item. When you start working on a part, you press START. When you finish, you press STOP. The system automatically records the time and stores it.

This information helps the production team understand how long jobs take, spot any problems, and plan the day more accurately.

💡
You do not need to be technical to use this system.

If you can use a smartphone, you can use this system. Everything is done with big, clearly labelled buttons.

User Roles

Every person who uses the system has a role. Your role decides what you can see and do. There are four roles:

👷
Operator
  • Start and stop your own timers
  • View your own completed jobs
  • Cancel your own timer within 60 seconds
🦺
Supervisor
  • Everything Operators can do
  • View all operators' history
  • Cancel or adjust any timer
  • Must provide a reason for changes
📊
Manager
  • Everything Supervisors can do
  • View the dashboard with stats
  • Export data to a spreadsheet
  • Filter by date, operator, item
🔧
Administrator
  • Everything Managers can do
  • Create and edit user accounts
  • Reset passwords
  • Enable or disable accounts
ℹ️
Not sure what role you have?

After you sign in, your name and role are shown in the top-right corner of every screen. If you need a different role, speak to your administrator.

1
Enter the Item Number

Tap the Item Number box and type the part number you are working on (for example, PHL-1001). Item numbers can contain letters, numbers, hyphens and underscores — no spaces. You can also scan a barcode instead of typing — see the next section.

⚠️
You can only have one timer running at a time.

If you try to start a second timer before stopping the first, you will see an error message. Stop your current job first, then start the new one.

Scanning a Barcode or QR Code

Instead of typing the item number, you can scan a barcode or QR code using the tablet's camera. This is faster and avoids typing errors.

Tap the Scan button

Next to the Item Number box, tap the small ⊞ Scan button. The camera screen will open.

2
Allow camera access (first time only)

The first time you scan, Chrome will ask if the website can use your camera. Tap Allow. You will not be asked again on this device.

💡
Torch button

If the barcode is in a dark area, a 🔦 Torch button will appear at the bottom of the camera screen. Tap it to switch on the tablet's flash as a light.

Cancelling a Timer

If you started a timer by mistake — for example, you scanned the wrong item number — you can cancel it. There are two situations:

Within 60 seconds of starting

If you realise the mistake quickly, tap the small Cancel button (next to the STOP button). A confirmation box will appear asking you to confirm. Tap Cancel Timer to confirm, or Keep Running if you changed your mind.

After 60 seconds

Once 60 seconds have passed, you cannot cancel your own timer. You will need to ask your supervisor to cancel it for you. This rule exists to keep the records accurate.

⚠️
Cancellations are recorded in the audit trail.

All cancellations — including who cancelled it, when, and the reason — are stored permanently. This is for quality and traceability purposes.

Supervisor: Cancelling or Adjusting a Timer

As a supervisor, you can cancel or adjust any timer — including timers that are over 60 seconds old. You must always enter a reason when making changes, as this is recorded in the audit trail.

To cancel an active timer

1
Find the active timer in History

Search for the operator's records. Active timers are shown with a green active badge.

2
Enter a reason

A box will appear asking you to explain why the timer is being cancelled. This is mandatory — you cannot proceed without entering a reason.

3
Confirm the cancellation

Tap Cancel Timer to confirm. The record is updated immediately.

🔒
All changes are permanently recorded.

Every cancellation and adjustment is stored in the audit trail along with your name, the date and time, and the reason you provided. This cannot be deleted.

Manager: Exporting Data to a Spreadsheet

You can download all the timing records as a CSV file — a format that opens in Microsoft Excel, Google Sheets, or any spreadsheet program.

1
Go to the Dashboard

Tap ☰ → Dashboard.

2
Set your filters

Use the From and To date pickers to choose the date range you want to export. You can also filter by a specific item number or operator if needed — leave blank to export everything.

3
Tap ⬇ Export CSV

The file will download to your device automatically. On an Android tablet it will appear in your Downloads folder.

4
Open the file in Excel or Google Sheets

The file contains one row per job with columns for item number, operator name, start time, end time, and duration in both seconds and minutes.

💡
The file opens correctly in Excel.

The CSV file includes a special character at the start that tells Excel to display times and dates correctly. You should not need to change any settings.

This is in the top-right of the User Management screen.

2

3

The account is created immediately. Tell the new user their username and password — advise them to treat the password as private.

Disabling an account

If a user leaves the company, tap Edit next to their name, untick the Account Active box, and tap Save Changes. They will no longer be able to sign in, but their history records are kept.

Administrator: Resetting a Password

If a user has forgotten their password, you can reset it for them.

1
Find the user in the User Management screen

Scroll through the list until you find the right person.

2
Tap Reset PW

A box will appear asking you to enter a new password for that user.

3
Type a new password and tap Reset Password

The password must be at least 8 characters long. Tell the user their new password — they can use it to sign in straight away.

Troubleshooting

I cannot sign in

  • Check your username and password are typed correctly
  • Usernames are not case-sensitive, but passwords are
  • After 10 failed attempts, wait 15 minutes before trying again
  • If your account may have been disabled, contact your administrator

The Scan button says the barcode library is not loaded

  • Make sure you are using Google Chrome on your Android tablet
  • Check that Chrome is up to date (open Chrome → tap the three dots → Help → About Chrome)
  • The scan feature requires Chrome version 83 or later (released 2020)
  • As an alternative, type the item number manually

The camera does not open when I tap Scan

  • You may have previously denied camera permission — tap the padlock icon in the Chrome address bar, find Camera, and change it to Allow
  • Then refresh the page and try again

My timer is still showing as active but I finished the job

  • Go to the Timer screen and tap STOP
  • If the timer is not visible, refresh the page
  • If it still will not stop, contact your supervisor — they can stop or cancel it for you

I accidentally started a timer for the wrong item

  • If less than 60 seconds have passed, tap Cancel on the active timer panel
  • If more than 60 seconds have passed, contact your supervisor

The page is not loading

  • Check the tablet is connected to Wi-Fi
  • Try pulling the page down to refresh
  • If the problem continues, the server may be restarting — wait 30 seconds and try again
📞
Still having trouble?

Contact your system administrator or supervisor. They can access the server logs and user settings to investigate further issues.
