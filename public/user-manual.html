What is Work Time?

Work Time is a shopfloor timing system built for Philtronics Ltd. When an operator starts working on an assembly job they press START. When they finish they press STOP. The system records exactly how long the job took, which workstation it was done on, and which work order it belongs to.

Supervisors and managers can see all active jobs in real time on the Wall Board, message operators directly, and view performance against target times. Managers can export data to a spreadsheet for further analysis.

💡
No technical knowledge needed.

If you can use a smartphone, you can use Work Time. Everything is done with large, clearly labelled buttons on a tablet screen.

User Roles

Every account has a role that controls what they can see and do.

👷
Operator
  • Start and stop own timers
  • View own job history
  • Cancel own timer within 60s
🦺
Supervisor
  • All Operator actions
  • Home dashboard
  • Wall Board (both views)
  • Message operators
  • View all history
  • Cancel/adjust any timer
📊
Manager
  • All Supervisor actions
  • Dashboard and stats
  • Target Times management
  • CSV export
  • Requires 2FA
🔧
Administrator
  • All Manager actions
  • Create and edit users
  • Reset passwords and 2FA
  • Delete timer records
  • Requires 2FA
ℹ️
Check your role.

After signing in your name and role appear in the top-right corner of every screen.

Signing In

Open Chrome on your tablet and go to the Work Time web address your supervisor gave you.

1
Enter your username

Not case-sensitive, e.g. operator1.

2
Enter your password

Characters appear as dots.

3
Tap Sign In

Operators go straight to the Timer screen. Supervisors and above go to the Home dashboard. Managers and administrators will also be prompted for a 2FA code if configured.

⚠️
10 failed attempts locks the device for 15 minutes.

Two-Factor Authentication (2FA)

Manager and Administrator accounts are required to use two-factor authentication for Cyber Essentials compliance. Operator and Supervisor accounts use standard password login only.

Setting up 2FA for the first time

After your first login a setup screen appears automatically.

1
Download an authenticator app

Google Authenticator or Microsoft Authenticator -- free from the App Store or Google Play.

2
Scan the QR code

Open the authenticator app, tap the + button, and scan the QR code shown on screen.

3
Enter the 6-digit code and tap Enable 2FA

The code changes every 30 seconds. Once confirmed, 2FA is active on your account.

Logging in with 2FA

After entering your username and password correctly, the screen switches to a Two-Factor Verification step. Open your authenticator app, find the Work Time entry, and type the 6-digit code shown. Tap Verify.

⚠️
Lost your phone?

Contact your administrator -- they can reset your 2FA so you can set it up on a new device.

Form Fields Explained

FieldRequired?What to enter
Item NumberRequiredThe part or assembly number. Can be scanned. Letters, numbers, hyphens and underscores only, max 40 characters.
WorkstationOptionalYour workstation or bench e.g. WS-01. Can be scanned. Appears in all history views, exports, and the Wall Board.
W/O NumberOptionalThe Work Order number e.g. WO-10042. Can be scanned. Appears in history and CSV exports for traceability.
Time CheckOptionalMarks this as a formally measured timing. Shows as a blue Time Check badge in all history views.

Scanning a Barcode or QR Code

Every field with a ⊞ Scan button can be filled by scanning a barcode or QR code with the tablet camera.

1
Tap the Scan button next to the field you want to fill
2
Allow camera access (first time only)

Tap Allow when Chrome asks. You will not be asked again on this device.

3
Point the camera at the barcode or QR code

The system reads it automatically within a second or two. Good lighting helps.

4
The field fills automatically

The camera closes. Check the value then continue as normal.

💡
Torch button.

In low light a Torch button appears at the bottom of the camera screen.

ℹ️
Timer restores automatically.

If you close the browser or switch devices, your active timer restores when you log back in. The elapsed time is calculated from the server so it is always accurate.

Target Time on the Timer Screen

If a manager has set a target time for the item you are working on, a progress bar appears beneath the running clock showing:

  • Green bar -- under 80% of target time elapsed
  • Amber bar -- between 80% and 100% of target (approaching)
  • Red bar -- target exceeded

The label shows the target time and either the time remaining (e.g. 17m 42s remaining) or how long overdue you are (e.g. 8m 15s overdue). This updates live every second.

If no target has been set for this item, the progress bar is not shown.

Cancelling a Timer

Within 60 seconds of starting

Tap the Cancel button next to STOP. A confirmation box appears -- confirm or tap Keep Running to go back.

After 60 seconds

Only a Supervisor or above can cancel. Ask your supervisor -- they must enter a reason which is permanently recorded in the audit trail.

🔒
All cancellations are permanently recorded.

Who cancelled, when, and the reason are stored and cannot be deleted.

Viewing Your History

Tap the menu (☰) then History to see your completed jobs. The default view shows the last 7 days. Each entry shows the item number, workstation, W/O number, start and end times, duration, and any Time Check or target time badges.

Messaging Operators

Supervisors and above can send a real-time message to any operator who is currently logged in. Messages appear instantly as a popup on the operator's screen.

On the Wall Board, tap the ✉ Message button at the bottom of any tile. On the Home dashboard, tap the ✉ icon next to the operator's name in the Active Jobs list.

1
Tap the Message button

A dialog opens with the operator's name.

2
Type your message

Up to 500 characters. Press Ctrl+Enter or tap Send Message.

3
The message appears instantly on the operator's screen

A popup notification shows your name, the message, and the time sent. It stays until they dismiss it or after 60 seconds.

ℹ️
Operator not logged in?

If the operator is not currently connected a message will confirm the message was not delivered.

Supervisor: Viewing All History

The History screen has additional filters for Supervisors -- you can search by Operator ID, Item Number, and Status. Leave fields blank to see all records.

💡
Finding stuck timers.

In the Status dropdown select Active (stuck timers) and tap Search. This removes the date filter so any timer still showing as active appears regardless of when it was started.

Supervisor: Cancelling or Adjusting a Timer

Supervisors can cancel or adjust any timer including those over 60 seconds old. A reason is always required and is permanently recorded in the audit trail.

  • Min / Max -- fastest and slowest recorded times
  • Target -- the set target time for this item (blank if none set)
  • Avg Delta -- the difference between average actual and target. Green if under target, red if over. Blank if no target is set.

Manager: Target Times

Tap the menu then Target Times to manage the target completion times per item number. You can also access this from the Dashboard page. Targets are visible to all roles on the Wall Board, timer screen, and history entries.

1
Tap + Add Target Time
2
Enter or scan the Item Number

The Scan button lets you scan the same barcode the operator uses, eliminating typing errors.

3
Set the target hours and minutes

For example 1 hour 30 minutes for a job that should take 90 minutes.

4
Tap Add Target Time

The target is saved immediately and will appear on all active tiles and timers for that item number.

To edit an existing target, tap Edit next to the item in the list. To remove a target entirely, tap the delete button.

Manager: CSV Export

On the Dashboard, set your date range and filters, then tap Export CSV. The file downloads to your device and opens in Excel or Google Sheets.

The export includes: Item Number, Operator, Workstation, W/O Number, Start and End times (UTC and London time), Duration in seconds and minutes, Time Check flag, and Status.

Administrator: Managing Users

Tap the menu then Admin to manage user accounts.

Adding a User and Resetting a Password

1
Tap + New User

Top-right of the User Management screen.

2
Fill in Username, Full Name, Password and Role

Username must be letters and numbers only. Password must be at least 8 characters.

3
Tap Create User

The account is created immediately. Tell the user their credentials.

To reset a password, find the user in the list and tap Reset PW. Enter a new password of at least 8 characters and tap Reset Password.

💡
Disabling an account.

Tap Edit next to the user, untick Account Active, and tap Save Changes. The user cannot log in but their history records are kept.

Resetting a User's 2FA

If a Manager or Administrator has lost their phone or authenticator app, find them in the User Management list and tap Reset 2FA. Confirm the action and they will be prompted to set up 2FA again on their next login.

🔒
Only reset 2FA when you are certain of the user's identity.

Resetting 2FA temporarily reduces account security until the user completes setup again.

Deleting a Timer Record

Go to History, find the record, and tap the delete button (🗑) on the right side of the entry. A confirmation box appears before anything is deleted.

⚠️
Deletion is permanent and cannot be undone.

The timer record and its full audit log are removed from the database permanently.

💡
Cancel All Stuck Timers.

In the Admin page there is an emergency tool to cancel all currently active timers at once. Use this only if multiple operators have stuck timers that cannot be stopped individually.

Troubleshooting

I cannot sign in

Check your username and password are typed correctly. Passwords are case-sensitive. After 10 failed attempts wait 15 minutes. If your account may be disabled, contact your administrator.

I am not being asked for my 2FA code

2FA is required for Manager and Administrator accounts only. If you have not yet set up 2FA you will see a setup prompt after login. If you have dismissed this with Remind Me Later you will be prompted again at your next login.

The Scan button is not working

Make sure you are using Google Chrome. If camera permission was denied, tap the padlock icon in the address bar, change Camera to Allow, then refresh. Chrome version 83 or later is required.

My active timer is not showing after a refresh or on a different device

Tap the menu then Timer. The system restores your active timer automatically from the server. If you see the error but no STOP button, sign out and back in.

The page is not loading

Check the tablet is connected to Wi-Fi. Pull the page down to refresh. Wait 30 seconds and try again if the problem continues.

I sent a message but the operator did not receive it

The operator must be currently logged in to receive messages. If they are not connected you will see a notification that the message was not delivered. Ask them to log in and try again.
